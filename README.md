# Cameron Moses — Frontend Software Engineer Portfolio v2.0.26

A sleek, fast, and responsive portfolio website designed to showcase modern web experiences, production-grade applications, and a robust frontend technical stack.

---

## 📄 Site Description

This repository contains the source code for my personal portfolio website. Built with a developer-centric, terminal-inspired aesthetic mixed with high-end editorial layouts, the site serves as a digital hub for my engineering journey. It bridges my past background in operations and client-facing roles at companies like Uber and Applied Materials with my current expertise as a full-time frontend software engineer. 

The application is fully optimized for performance, modular component architecture, and fluid user experiences, serving as a live demonstration of my design-to-code capabilities.

---

## ✨ Key Features

* **Interactive Terminal Component:** A custom, interactive-style React representation of my developer profile, current focus, and core interests.
* **Selected Project Showcase:** A curated display of production-ready applications featuring detailed breakdowns, tech tags, and direct links to live demos and repositories (including Skinstric, Summarist, and Ultraverse NFT Market).
* **Dynamic UI/UX Animations:** Meticulously crafted UI transitions and micro-interactions that elevate the editorial aesthetic without compromising on performance.
* **Comprehensive Experience Timeline:** A clean historical breakdown detailing the transition from real estate and advisory consulting into engineering through the Frontend Simplified track.
* **Contact & Network Integration:** Quick-access operational links to downloadable assets (Résumé PDF), GitHub, LinkedIn, and direct email channels.

---

## 🛠️ Tech Stack

The architecture of this portfolio relies on modern web technologies selected for type safety, maintainability, and rapid rendering:

### Core Frameworks & Languages
* **React & Next.js (v15):** Leveraging modern component structures and server-ready architecture.
* **TypeScript:** Ensuring robust type safety, predictable data flows, and scalable code maintenance.
* **HTML5 & CSS3:** Semantically structured layouts paired with modern styling methodologies.

### Styling & Animation
* **Tailwind CSS:** For clean, responsive, and utility-first styling patterns.
* **Framer Motion / Advanced Animation Systems:** Delivering smooth, hardware-accelerated interactive elements and layout transitions.

### Services & Tools
* **Git & GitHub:** Employed for strict version control, descriptive commit messaging, and repository management.
* **Vercel:** Utilized for seamless continuous integration and high-performance edge deployment.
